Demo Video Folder
=================

This folder is for storing demo videos for the School ERP System.

Instructions:
1. Place your demo video file here (e.g., demo-video.mp4, demo-video.webm, etc.)
2. Update the video path in resources/views/welcome.blade.php
3. Supported formats: MP4, WebM, OGG

Example file names:
- demo-video.mp4
- school-erp-demo.mp4
- system-demo.webm

The video will be displayed in the "See It In Action" section of the welcome page.
